# 🎨 Basic CSS Projects Collection

This repository contains a collection of beginner-friendly projects built using **HTML5** and **CSS3**. 
These projects helped me practice core web development concepts such as layout design, typography, forms, tables, animations, and responsive design.

---

📁 Projects Included

### 1. Build a Survey Form
A responsive survey form with various input fields like text boxes, radio buttons, checkboxes, and dropdown menus.

### 2. Tribute Page
A tribute page dedicated to a famous personality with images, biography, and achievements.

### 3. Web Art Page
A creative design page built using only HTML and CSS shapes, gradients, and positioning.

### 4. Parallax Website
A website demonstrating the parallax scrolling effect using background images and CSS properties.

### 5. Technical Documentation Page
A structured documentation layout with a sidebar navigation and content sections.

### 6. Developer Landing Page
A landing page designed for developers to showcase skills, services, and contact information.

### 7. MyTunes Website
A clone of the MyTunes landing page with multiple sections and responsive layout.

### 8. Job Application Form
A professional-looking job application form with validation-friendly input fields.

### 9. Sweet Shop Project
A colorful website for a sweet shop featuring products, banners, and navigation.

### 10. BioData
A simple biodata/resume page displaying personal and educational details.

### 11. Periodic Table Design
A complete periodic table created using CSS Grid and custom styling.
